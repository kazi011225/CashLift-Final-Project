# CashLift

An AI-powered donation app that explains the real-world impact of your donation
in plain language, to fight poverty with clarity instead of vague promises.

## Status
🚧 Work in progress — this README will be filled out fully (features, schema,
run instructions, API used) as the project is built.

## Tech stack
- Node.js + Express
- EJS templating
- MongoDB (via Mongoose)
- Google Gemini API

## Running locally
1. `npm install`
2. Copy `.env.example` to `.env` and fill in your own values
3. `npm start`
4. Visit `http://localhost:3000`
